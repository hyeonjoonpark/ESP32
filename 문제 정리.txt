ESP32보드에 가상의 온습도센서가 연결된다
upload.php로 GET방식으로 센서밧을 투척하면 데이터베이스에 저장된다
(서버가 데이터를 수집한 시간도 저장)
URL의 예시는 아래와 같다!

/upload.php?did=device1&temp=1.34&humi=50.11

did : 디바이스 아이디
temp : 온도
humi : 습도

---------------------------------------------------------------------------------------------
DB의 이름은 bssm2_4로 만들고 sensor table을 아래와 같이 만드시오

num : key값 int AI
did : 디바이스아이디 varchar(20),
temp : 온도값 float
humi : 습도값 float
date : 업로드시간


-------------------------------------------------------------------------------------

view.php를 만들어서 sensor table 안에 있는 데이터를 모두 출력하시오

------------------------------------------------------------------------------------

device.php를 만들어서 디바이스의 정보를 입력받고,
device_process.php에소 데이터베이스에 저장하시오.

이때 디바이스의 정보는 device table에 저장하시오

did : 디바이스아이디 varchar(20) primary key
loc : 설치장소 varchar(100)
type : 센서종류 varchar(20)
date : 설치날짜 varchar(20)

--------------------------------------------------------------------------------------

upload.php를 device table에 등록된 것이 아니면 저장하지 마시오

---------------------------------------------------------------------------------------

view.php 에 내가 원하는 디바이스명에 해당하는 센서데이터만 출력되도록 하시오
